## ‚úÖ Validation Checklist

Before sending to your API, ensure:

- [ ] `title` is 10-100 characters
- [ ] `slug` is lowercase, no spaces, only letters/numbers/hyphens
- [ ] `content` is valid HTML with `<div class='post-content'>` wrapper
- [ ] `excerpt` is 100-250 characters
- [ ] `category_id` exists in your categories table
- [ ] `featured_image` is a valid URL (or omit if none)
- [ ] `tags` is an array of strings
- [ ] `status` is "published", "draft", or "archived"
